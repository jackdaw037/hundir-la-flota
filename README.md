# hundir-la-flota

Este proyecto es una implementación del clásico juego Hundir la Flota / Batalla Naval para dos jugadores, desarrollado en Java y jugado completamente desde la consola.

Los jugadores colocan sus barcos en un tablero de 10x10 y se turnan para disparar hasta que uno de ellos hunde todos los barcos del rival.

Características del juego

Juego para 2 jugadores

Tablero de 10x10

Colocación manual de barcos

Turnos alternos

Sistema de disparos con:

X = Tocado

O = Agua

- = Casilla no descubierta

Detección automática del ganador
